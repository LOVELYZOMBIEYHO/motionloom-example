# Card Animation Layout

A floating card layout scene with staggered entry timing and editorial UI styling.

## Teaches

- Group-based card components with rounded rectangles and text.
- Staggered opacity expressions for sequential reveal.
- Lightweight UI motion graphics using only scene primitives.

## Run

```bash
cargo run -p motionloom --example render_file_frame -- \
  showcase/s-000021/main.motionloom /tmp/frame.png 0 cpu
```
