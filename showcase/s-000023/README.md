# Ease In Out Bezier Curve Demo

A minimal curve timing comparison using two animated line endpoints.

## Teaches

- Named `ease_in_out` timing.
- Custom `ease(...)` cubic Bezier timing.
- Curve-driven line endpoint animation.

## Run

```bash
cargo run -p motionloom --example render_file_frame -- \
  showcase/s-000023/main.motionloom /tmp/frame.png 0 cpu
```
