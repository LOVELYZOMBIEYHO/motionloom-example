# Animated Path And Polyline

A compact scene comparing animated polyline and Bezier path stroke reveals.

## Teaches

- Polyline route animation.
- Bezier path trim animation.
- Labels and endpoint markers for teaching path concepts.

## Run

```bash
cargo run -p motionloom --example render_file_frame -- \
  showcase/s-000022/main.motionloom /tmp/frame.png 0 cpu
```
