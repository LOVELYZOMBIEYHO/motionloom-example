# Business Bar Chart Animated

A business-style quarterly revenue chart with animated bars and reusable gradient definitions.

## Teaches

- Chart panel, grid, axis labels, and grouped bar components.
- Reusable gradient definitions for consistent business graphics.
- Curve-driven bar height and label reveal animation.

## Run

```bash
cargo run -p motionloom --example render_file_frame -- \
  showcase/s-000020/main.motionloom /tmp/frame.png 0 cpu
```
