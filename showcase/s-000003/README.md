# Rainbow Spin 4s

A simple MotionLoom motion-graphics showcase: 7 rainbow-colored circles orbiting around a dark center, rotating continuously over 4 seconds.

Use in Anica:

1. Open the MotionLoom / VFX page.
2. Open `main.motionloom`.
3. Paste the full DSL into the MotionLoom editor.
4. Preview or render.

Notes:

- Scene-only DSL; no Process or World graph nodes are used.
- All 7 circles remain at full opacity throughout.
- The group rotation uses `ease_in_out` easing for smooth acceleration and deceleration.
- Seven colors used: red (#FF3B30), orange (#FF9500), yellow (#FFCC00), green (#34C759), cyan (#5AC8FA), blue (#007AFF), purple (#AF52DE).
