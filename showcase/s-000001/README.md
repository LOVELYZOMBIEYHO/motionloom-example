# ANICA Cyberpunk Promo 20s

Scene-only MotionLoom DSL version of the 20-second ANICA concept promo.

Use in Anica:

1. Open the MotionLoom / VFX page.
2. Open `main.motionloom`.
3. Paste the full DSL into the MotionLoom editor.
4. Preview or render.

Notes:

- Audio and SFX are not generated by this DSL file.
- World and Process are represented visually as Scene graphics and text, not as live mixed World/Process graph execution.
