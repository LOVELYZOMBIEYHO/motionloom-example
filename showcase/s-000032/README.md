# s-000032 Fearless RGB Glitch Typography

Original RGB glitch typography promo built from fixed text layers and numeric motion.

- Duration: `9s`.
- Canvas: `1920x1080`.
- Build RGB glitch typography by layering offset Text nodes.
- Use deterministic noise, scanlines, and slice rectangles for a digital glitch look.
- Sequence motivational text beats with simple opacity and scale curves.

Run:

```bash
cargo run --release -p motionloom --example wgpu_live_preview -- ../motionloom-example/showcase/s-000032/main.motionloom
```
