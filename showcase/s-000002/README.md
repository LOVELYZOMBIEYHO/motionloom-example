# Aged Magic Book Title Reveal 7s

Polished MotionLoom showcase for an original fantasy parchment title reveal. The scene uses layered paper texture, ink clouds, restrained bloom, and a page-turn transition into a second title card.

Use in Anica:

1. Open the MotionLoom / VFX page.
2. Open `main.motionloom`.
3. Paste the full DSL into the MotionLoom editor.
4. Preview or render with GPU output for the intended bloom look.

Notes:

- This is an original fantasy-book style scene; no licensed logo or title text is reproduced.
- The graph combines a Scene source with a Process bloom pass.
- CPU render works for checks, but GPU render/export is the intended path for performance.
