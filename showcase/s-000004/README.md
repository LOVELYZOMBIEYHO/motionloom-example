# ANICA Typewriter Float 8s

Scene-only MotionLoom DSL version of an 8-second vertical title sequence with typewriter-style text reveals.

Use in Anica:

1. Open the MotionLoom / VFX page.
2. Open `main.motionloom`.
3. Paste the full DSL into the MotionLoom editor.
4. Preview or render.

Notes:

- Audio and SFX are not generated by this DSL file.
- Three sequential words: **ANICA** → **VFX** → **MOTION** with typewriter cursor animation.
- Each word has a ghost/outline version behind the main text.
- Features: floating motion, starfield particles, scanlines, ambient glow, and horizontal cut line.
- Vertical format (1080x1920) optimized for mobile/portrait display.
