# DSL Text Basic

ID: `cs-000021`  
Type: `core`  
Domain: `scene`

## Features

text, text-render-scale

## DSL

text, renderScale

## Teaches

- Use the basic <Text> pattern in MotionLoom.
- Use `renderScale="4x"` to rasterize text at four times the pixel density without changing layout size.

## Use In Anica

Open `main.motionloom`, copy all content, then paste it into the Anica VFX / MotionLoom page.
