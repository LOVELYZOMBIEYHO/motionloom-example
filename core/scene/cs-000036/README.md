# AnimationTarget Keyframe MVP

This core scene demonstrates AE-style property animation using top-level
`AnimationTarget` blocks.

## What it demonstrates

- Targeting a node by `id`.
- Animating a single property with frame-based `Key` nodes.
- MVP properties: `x`, `y`, `rotation`, `scale`, and `opacity`.
- Per-frame evaluation without writing a long `curve(...)` string.
