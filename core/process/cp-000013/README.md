# CP-000013 — Fixed Opacity Layer FX

This example shows the smallest normal Layer FX process for fixed opacity.

Use `main.motionloom` in Anica Layer FX. It expects the selected layer as
`input:clip0`, converts it into a process texture, applies `effect="opacity"`,
and presents the result.

Parameters:

- `opacity`: `0.0` to `1.0`; this example uses `0.3`.

`main_with_scene.motionloom` is a standalone preview version with a built-in
scene source.
