# Scene World Process Composition Basic

ID: `cm-000001`  
Type: `core`  
Domain: `composition`

## Features

scene, world, process, text, shape, actor, material, gpu-pipeline

## DSL

Graph, Background, Scene, Timeline, Track, Sequence, Layer, Rect, Circle, Text, World, Camera, Actor, Material, Process, Tex, Pass, Present

## Teaches

- Mix Scene, World, and Process graph families in one composition graph.
- Use Scene and World outputs as Process textures, then present the final Process from the Graph root.
- Keep one root-level <Present> node as the final child of <Graph>.
