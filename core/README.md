# Core Examples

Core examples are learning-first MotionLoom examples. They are organized by graph domain:

- `scene/`: Scene, Timeline, Track, Sequence, Layer, text, shape, image, SVG, video, and 2D animation examples.
- `world/`: World, Camera, Actor, Material, Action, Bone, Pose, and 3D asset examples.
- `process/`: Process, Input, Tex, Pass, Present, and GPU pipeline examples.
- `composition/`: mixed Scene + World + Process examples only.
- `text/`: Text-specific typography, boxed labels, and reusable text style examples.

Use `../browse/` for generated lookup by domain, feature, and DSL tag.
