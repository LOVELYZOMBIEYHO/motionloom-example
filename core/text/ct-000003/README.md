# Gold Serif Extrude

ID: `ct-000003`  
Type: `core`  
Domain: `text`

## Style

## Features

- Combine duplicate `Text` nodes and simple primitives for depth, glow, or editorial styling.
- Keep the style reusable as a MotionLoom text preset.
