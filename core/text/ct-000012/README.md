# CT-000012 — Text AA, Soft Edge, and Blur

Demonstrates MotionLoom text edge controls:

- `renderScale="auto"` for default supersampling.
- `antialias="grayscale"` / `antialias="subpixel"` as smoothing presets.
- `edgeSmoothing` / `softEdge` for alpha-edge softening.
- `blur` for an obvious animated blur pulse from `0px` to `14px` and back.
